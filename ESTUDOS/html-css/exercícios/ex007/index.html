Observação VS Code!

Adicionando uma tag no texto selecionado:

Para adicionar uma tag há um texto sem ter que remove-lo e depois incluí-lo dentro da tag, você pode selecionar o texto em questão e pressionar as teclas:
Ctrl+Shift+P que irá abrir uma linha de pesquisa.

Nela, você irá buscar por Envelope com abreviatura, ou somente abb e selecionar essa opção. Após isso, é só informar a tag que você quer para o texto ser inserido dentro desta tag.

Observação!

O texto grande trabalha com a tag <big></big>, que é uma tag descontinuada.
Pois se caso você quiser deixar um texto maior, estamos falando de estilo.

O texto pequeno trabalha com a tag <small></small>, e não é descontinuada.
Um exemplo claro de significado para essa tag, é em uma contrato, que pode possuir informações em letras pequenas.

Desta forma, existe as duas possibilidades de texto, entretanto, a tag BIG já se tornou obsoleta.
