Informação Importante:

Quando for utilizar a tag A, é recomendado que adicione o parâmetro target com o valor _blank, se não o fizer, quando o usuário clicar no link ele irá para o próximo site e não retornará.
Caso adicione esse parâmetro, ao clicar no link será aberto uma nova aba que vai carregar o link em questão.
Adicionar também o parâmetro rel (Relativo) com o valor external se for realmente um link externo.
